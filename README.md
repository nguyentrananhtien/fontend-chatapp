# Sử dụng React + Vite
Mẫu này cung cấp thiết lập tối thiểu để React hoạt động trong Vite với HMR và một số quy tắc ESLint.
Hiện tại, có hai plugin chính thức:
@vitejs/plugin-react sử dụng Babel (hoặc oxc khi được sử dụng trong rolldown-vite) cho Fast Refresh
@vitejs/plugin-react-swc sử dụng SWC cho Fast Refresh
Trình biên dịch React
Trình biên dịch React không được bật trên mẫu này vì ảnh hưởng của nó đến hiệu suất phát triển và xây dựng. Để thêm nó, hãy xem tài liệu này.
# Mục cập nhật tiến độ làm việc của nhóm:
- Tuần đầu bắt đầu tìm hiểu sâu về chat app cũng như api, viết những trang cũng như các chức năng cần thiết như đăng kí và đăng nhập (sử dụng call api có sẵn trong link)
- Tuần thứ 2 sửa lại dự án theo hương ReactJs and Vite và đã có các trang cơ bản như phần login,res,chat
